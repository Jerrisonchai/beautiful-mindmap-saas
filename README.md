# Beautiful Mindmap SaaS

> Turn ideas into beautiful mindmaps in seconds, not hours.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to see the app.

## 📁 Project Structure

```
├── docs/          # Product documentation (PRD, specs)
├── tasks/         # Development tasks for AI agents
├── prompts/       # AI agent prompts
├── src/           # Source code
│   ├── app/       # Next.js App Router
│   ├── components/# Reusable UI components
│   └── lib/       # Utilities
└── tests/         # Test suite
```

## 🛠️ Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Visualization:** D3.js / React Flow (coming soon)
- **Database:** PostgreSQL (Supabase) (coming soon)
- **Auth:** Clerk / NextAuth.js (coming soon)

## 📋 Development Workflow

1. Read `docs/prd.md` for product requirements.
2. Check `tasks/` for current development tasks.
3. Run AI agents with prompts from `prompts/`.
4. Commit changes with clear messages.
